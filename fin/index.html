Merci d'avoir utilisé CNRTLReader.

Plusieurs projets ont développé des alternatives à cette application. Veuillez les utiliser et y contribuer plutôt que celui-ci:

  • BetterCNRTL par AndroneDev

Le code source de CNRTLReader est toujours disponible sur GitHub.
